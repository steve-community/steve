\subsection{Ausführliche Beschreibung}
There are some implementations of S\-O\-A\-P/\-W\-S-\/\-A, which leave the required Message\-I\-D element out. This behaviour is incorrect! Since C\-X\-F throws an exception in absence of this element, we provide a workaround which adds this S\-O\-A\-P header to the incoming message.
