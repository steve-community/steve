boolean de.rwth.idsg.steve.repository.impl.OcppServerRepositoryImpl.updateChargebox ( UpdateChargeboxParams  params)

One DB call with two functions:

  1. Update all fields with the exception of chargeBoxId, so that initially it is enough to register a chargebox with its ID in DB. During boot, the chargebox provides missing information which might be updated (for ex: firmware)
  2. If the chargebox not registered => no chargeboxes to update => updated/returned row count = 0
