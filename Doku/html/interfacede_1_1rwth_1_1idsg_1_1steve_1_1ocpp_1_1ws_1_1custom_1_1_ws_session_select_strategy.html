WebSocketSession getSession (Deque< SessionContext > sessionContexts)
 

Ausführliche Beschreibung

We want to support multiple connections to a charge point. For sending messages we need a mechanism to select one WebSocketSession. Implementations of this interface should use different mechanisms to realize that.
